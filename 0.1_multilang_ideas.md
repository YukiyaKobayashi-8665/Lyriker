Idea: develop the app to have the following capabilities:
  1) Edit and display the translation of the lyrics alongside the original lyrics.
  2) Edit and display notes on the original lyrics, including vocabs and grammar.

0. Updates to the original lyrics display
   1. The user should be able to divide the lyrics into chunks(verse-1, chorus-1, verse-2, etc.). 
      1. The stamp mode should support chunking the lyrics.
   2. The lyrics display should show the lyrics by chunks
      1. During playing of a song, the active chunk is displayed with a larger font, while the inactive chunks above and below are shown with smaller font and higher opacity.
1. Translation section
   1. The UI should have a translation section to display and edit translation
      1. The display should show the translation in chunks, in a similar way as the lyrics display
      2. The translation section should support editing the translation similar to the way lyrics display does.
      3. The translation section should support chunking the translation, and binding the timestamps of the translation chunks to the lyrics chunks. The translation section does not need to support line-by-line timestamping.
2. Notes section
   1. The UI should have a notes section to display and edit notes on the vocab and grammar of the current chunk of lyrics
   2. The notes section only shows notes related to the current active chunk of lyrics.
   3. The lyrics section should have a button to create notes or edit notes on the current section.
